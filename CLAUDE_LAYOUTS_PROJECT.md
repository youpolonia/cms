## 2. KLUCZOWE ZASADY

### Obrazki - KRYTYCZNE!
- **Media Library czyta z bazy danych media**, NIE skanuje folderu
- Zewnętrzne URL (Unsplash, Pexels) NIE działają na opublikowanej stronie
- **ZAWSZE** używać przycisku Save przy Image URL aby pobrać na serwer
- Lub wybierać obrazki z Media Library / Stock Photos / AI Generate

### Przycisk Save (dodany 2025-12-26)
- Pojawia się gdy Image URL zaczyna się od http
- Pobiera obrazek na serwer (/uploads/media/)
- Automatycznie dodaje do tabeli media
- Obrazek od razu widoczny w Media Library
