# Code Agent Execution Prompt

Execute the following 50 tasks in sequence for the CMS development:

1. Analyze current codebase and architecture
2. Break down each roadmap task into technical requirements
3. Create implementation plans for each feature
4. Set up task tracking system
5. Implement version control strategy
6. Establish CI/CD pipeline
7. Configure development environment
8. Implement core CMS features (1-15)
9. Develop MCP server enhancements (16-25)
10. Optimize performance and security (26-35)
11. Improve user experience (36-45)
12. Set up deployment systems (46-50)

## Execution Guidelines:
- Work sequentially through each task
- Create detailed technical specifications
- Implement features in small, testable increments
- Document all changes
- Maintain backward compatibility
- Include automated tests
- Optimize for performance
- Follow security best practices
- Provide progress reports after each task

## Success Criteria:
- All features implemented as specified
- Passing test coverage (>80%)
- Documentation complete
- Performance benchmarks met
- Security audits passed
- Successful deployment to staging
