## Managing Content Types
### Creating a New Content Type
1. Click "Add Content Type"
2. Enter:
   - Human-readable name
   - Machine name (auto-generated but editable)
   - Description (optional)
3. Click "Save"

### Editing an Existing Content Type
1. Locate content type in list
2. Click "Edit" button
3. Modify fields as needed
4. Click "Save"

## Managing Fields
### Adding Fields
1. While editing a content type:
2. Click "Add Field"
3. Configure:
   - Field name
   - Field type (text, number, select, etc.)
   - Required status
   - Default value (if applicable)
4. Click "Save"

### Reordering Fields
1. Use drag handles in field list
2. Or specify weight values manually
3. Changes save automatically
