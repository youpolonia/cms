# Search Management Guide

## Accessing Search Filters
1. Navigate to Admin Dashboard > Content > Search
2. Click "Advanced Filters" button to expand filter options

## Available Filters

### Content Type Filter
![Content Type Filter](placeholder://filter-content-type.png)
- Dropdown with all content types
- Multi-select enabled
- Default: All types selected

### Archive Status Filter
![Archive Status Filter](placeholder://filter-archive.png)
- Toggle between:
  - Active only
  - Archived only
  - Both
- Default: Active only

### Date Range Filter
![Date Range Filter](placeholder://filter-date.png)
- Calendar picker for:
  - Created date
  - Modified date
- Relative date options (Last 7 days, Last month, etc.)

## Saving Filter Presets
1. Configure desired filters
2. Click "Save Preset" button
3. Enter preset name
4. Choose visibility:
   - Private (only you)
   - Team (all admins)
5. Click "Save"

## Performance Tips
- Avoid overly broad date ranges
- Combine filters for better performance
- Use presets for frequently used combinations
- Archived content searches are slower - limit when possible

## Troubleshooting
- **No results**: Check if filters are too restrictive
- **Slow searches**: Narrow date range or content types
- **Missing filters**: Verify admin permissions
