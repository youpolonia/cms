# Keyboard Shortcuts

## Version Comparison & Restoration

| Shortcut | Action | Available In |
|----------|--------|--------------|
| `Ctrl+Alt+D` | Open version comparison dialog | Content Editor |
| `Ctrl+Shift+D` | Toggle detailed/semantic diff view | Version Comparison |
| `R` | Restore currently viewed version | Version Comparison |
| `Esc` | Close version comparison dialog | Version Comparison |
| `←` / `→` | Navigate between diff chunks | Version Comparison |
| `↑` / `↓` | Scroll through diff content | Version Comparison |

## General Navigation

| Shortcut | Action | Available In |
|----------|--------|--------------|
| `Ctrl+S` | Save current content | All Editors |
| `Ctrl+Shift+S` | Save as new version | All Editors |
| `Ctrl+Z` | Undo | All Editors |
| `Ctrl+Y` | Redo | All Editors |
| `Ctrl+F` | Find in content | All Editors |

## Accessibility Notes

- All shortcuts can be customized in user preferences
- Screen reader users can access all functionality via keyboard
- Shortcut conflicts are automatically detected and resolved
- Visual indicators show available shortcuts in context
