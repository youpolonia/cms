# CMS Deployment Instructions

## Required Permissions
- Directories: 755 (drwxr-xr-x)
- Files: 644 (-rw-r--r--)
- Special directories (cache, uploads): 777 (drwxrwxrwx)

## First-Run Setup
1. Upload all files to your web server
2. Rename config_sample.php to config.php
3. Edit config.php with your database credentials
4. Access /admin in your browser to complete setup

## Default Credentials
- Admin URL: /admin
- Username: admin
- Password: changeme (change immediately after first login)

## Security Recommendations
1. Change default admin credentials
2. Restrict access to /admin directory via .htaccess
3. Disable directory browsing in .htaccess
4. Regularly backup your database

## Troubleshooting
- 500 Errors: Check PHP error logs and file permissions
- Database Issues: Verify credentials in config.php
- White Screen: Enable debug mode in config.php
